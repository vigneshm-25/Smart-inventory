# 📦 Smart Inventory AI

**An AI-powered inventory and resource management platform for educational institutions and organizations.**

Smart Inventory AI modernizes how shared resources such as cameras, projectors, speakers, sports equipment, laptops, and other assets are managed. By combining **QR-based tracking**, **Artificial Intelligence**, **real-time updates**, and **voice interaction**, the platform simplifies inventory operations and helps organizations make smarter decisions.

## ✨ Features

* 📦 Smart inventory management
* 📱 QR-based borrow & return system
* 🤖 AI Inventory Assistant (chat with your inventory)
* 🎤 Voice-enabled inventory operations
* 🧠 Intelligent equipment planner for events
* 📊 AI-generated reports and analytics
* 📈 Real-time dashboard with live inventory status
* 🔔 Automated email/Telegram notifications
* ⭐ Member reliability scoring
* ⚡ Real-time synchronization using WebSockets

## 🛠️ Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** FastAPI
* **Database:** PostgreSQL
* **AI:** OpenAI GPT-5-mini
* **Authentication:** JWT
* **QR:** html5-qrcode, qrcode
* **Real-time:** WebSockets
* **Notifications:** Email, Telegram

## 🔄 Workflow

1. Admin registers an inventory item.
2. System generates a unique QR code.
3. Member scans the QR code to borrow or return the item.
4. Inventory updates instantly across the dashboard.
5. AI Assistant answers inventory queries and performs actions.
6. Equipment Planner recommends resources based on event details.
7. AI generates reports and insights.
8. Notifications remind users about upcoming or overdue returns.

## 🚀 Why Smart Inventory AI?

Unlike traditional inventory systems that only track assets, **Smart Inventory AI** acts as an intelligent assistant that helps organizations manage resources, plan events, generate reports, and make data-driven inventory decisions through AI.

**Built for smarter, faster, and AI-driven inventory management.**
